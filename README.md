    - `for` loops
    - arrays
      - `.push` & `.pop`

      - `.shift` & `.unshift`
      - `.slice` & `.splice`
      - `.split` & `.join`
      - `.reverse`
      - `.indexOf`
      - iteration over an array using a `for` loop
      - `.forEach`
    - objects
      - key value pairs
      - dot notation
      - bracket notation
      - iteration with `Object.keys()`
      - methods
      - `delete`
    - functions
      - parameters and arguments
      - scope
      - anonymous functions
      - `return`
